# 추천 시스템 구축에서의 이슈

## 콜드 스타트 문제

- 신규 사용자의 경우 베스트 셀러 추천
- 신규 아이템의 경우 추천 리스트에 포함

## 확장성

- 병렬처리 기술 : 스파크
- 배치 스케줄
- 알고리즘 변경
- 샘플링 모델 학습

## 추천의 활용

- 직접적 추천 리스트 제공
- 간접적 추천 리스트 제공
- 검색 결과 정렬
- 맞춤형 광고 제공
